# スライド生成デモ

テーマ指定から，単体HTMLのプレゼン資料を作るための作業用リポジトリ．

## 現在の成果物

- メインスライド: `outputs/ポケモン現環境最強構築_スライド.html`

## このリポジトリの使い方

1. テーマを決める．
2. 必要に応じて最新情報を確認する．
3. `outputs/` 配下に単体で開けるHTMLスライドを作る．
4. 修正は既存ファイルを更新し，差分はgitで管理する．

## 今回の題材

- テーマ: `ポケモン現環境最強構築`
- 前提: `ポケモンSV シングル / レギュレーションI / 2026-03-19時点`

## 運用方針

- 過去版のHTMLは複製保存しない．履歴はgitコミットで追う．
- スライドの内容は，フォルダ構造や作業用ディレクトリではなく，指定テーマそのものに集中して作る．
- 単なるテキスト羅列ではなく，画像，余白，視線誘導，情報の重心まで含めて完成度を上げる．
